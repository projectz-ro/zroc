◆ zroc_err_get()

ZrocErr * zroc_err_get ( void )

Get the thread-local error object.

If the thread does not have an error object yet, one will be allocated and initialized with code ZROC_OK and an empty message.

Returns
Pointer to the thread-local ZrocErr object.

◆ zroc_err_set()

void zroc_err_set ( ZrocErrCode code,
const char * msg )

Set the thread-local error.

Updates the thread-local error code and message. If msg is NULL, the default message for the given code from ZROC_ERRORS will be used.

Parameters
codeThe error code to set.
msgOptional error message. If NULL, the default is used.
